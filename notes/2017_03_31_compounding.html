Compounding Interest

31st March, 2017

We will begin with the basic equations of going forward in time (using percentage units): 
# compounding.py

from math import exp, log

# FUNCTIONS SHOULD RETURN NUMBERS, FORMAT SEPARATELY

def simple_compounding(rate, times, pv=100):
    return "{:.4f}".format(pv * (1 + (rate/100)) ** times)

def continuous_compounding(rate, times, pv=100):
    return "{:.4f}".format(pv * exp((rate/100) * times))
A useful conversion is taking a long period, such as a year, and finding the equivalent monthly rate that when compounded simply, will result in the original rate. 
def divide_rate(rate, times):
    fv = 1 + (rate/100)
    r = log(fv) / times
    return "{:.6f}%".format((exp(r) - 1) * 100)
Suppose I have an investment yielding 10% in a year. What would be the equivalent monthly rate if the investment were to be compounded monthly? 
>>> divide_rate(10, 12)
'0.797414%'
>>> simple_compounding(0.797414, 12)
'110.0000'
>>> (1+0.00797414) ** 12
1.0999999943832588
