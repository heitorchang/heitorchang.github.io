Cogite Python
Como pensar como um cientista da computação
Sobre esta tradução

Cogite Python é uma tradução do livro Think Python de Allen Downey, segunda edição (versão PDF 2.2.15).

O conteúdo de Cogite Python está escrito em Markdown e está disponível neste repositório do Github.

O HTML é criado por Doxygen v. 1.8.11 e está hospedado no Github Pages.

Tradução em português de Heitor Chang <heitor at alumni.stanford.edu>
