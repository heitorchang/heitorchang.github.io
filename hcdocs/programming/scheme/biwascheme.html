Notes

Warnings

  • After running (http-request "..."), stack trace in errors stop appearing.
    • Partial solution: in top-level HTML file, evaluate (define my-text (http-request "path.txt")) then refer to my-text inside script.scm and evaluate (load "script.scm")

General usage

  • The empty list, printed as (), is the null pointer value.
  • null? checks a value for ()
  • It must be written as '()
  • eq? checks if two things are the same object.
  • quote doesn't create a new data structure each time it's called, so structures created with quote should not be modified.
  • On the other hand, list does create new data structures.

Undocumented functions

  • (dom-element "#selector") is equivalent to document.getElementById("...").
    • Using BiwaScheme's documented functions, it is: (js-ref (getelem "#selector") 0)
  • (getelem "#selector") is ($ "#selector").

Custom functions

  • (element-set-value! selector value) sets the value of the selected element.

(define (date2runtime date)
  ; HACK
  ; wraps around occasionally!
  (+
     (* (date-hour date) 60 60 1000)
     (* (date-minute date) 60 1000)
     (* (date-second date) 1000)
     (date-millisecond date)
  )
)
