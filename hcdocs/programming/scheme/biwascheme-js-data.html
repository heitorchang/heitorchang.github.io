BiwaScheme reading JS data

There are four conversion functions:

  • (js-array->list)
  • (list->js-array)
  • (js-obj->alist)
  • (alist->js-obj)

To "initialize" a value: 

const listdata = [1,2,4,9,16]
(define js-listdata (js-eval "listdata"))
(define listdata (js-array->list js-listdata))

const jsondata = {"key": 123}
(define js-data (js-eval "jsondata"))
(define data (js-obj->alist js-data))  ; (("key" . 123))

const listobjs = [{"name": "A", "age": 12}, {"name": "B", "age": 23}]
(define listobjs (js-array->list (js-eval "listobjs")))

(js-obj->alist (car listobjs))
