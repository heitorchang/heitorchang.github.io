C# notes

The "new" .NET Core consists of:

  • CLR (Common Language Runtime) became CoreCLR
  • BCL (Base Class Library) became CoreFX

.NET MAUI, a cross-platform platform for mobile and desktop apps was released in May 2022.

The C# Compiler (Roslyn)

C# source code is converted to IL (Intermediate Language), stored in a DLL or EXE file. It is then executed by the CoreCLR Virtual Machine.

Xamarin was bought by Microsoft. It is a platform for mobile apps.

Info about Local SDKs and runtimes

Run the following to get info about your dev environment: 

dotnet --list-sdks
dotnet --list-runtimes
dotnet --info
