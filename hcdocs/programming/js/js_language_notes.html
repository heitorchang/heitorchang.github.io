JavaScript language notes

Type conversion

Use Number("3") to convert a value to a number. Alternatively, use parseInt() and parseFloat().

Variable declarations

Bindings declared with let and const are local to the block that they are declared in. Old-style var bindings are visible throughout the whole function they appear in (or global scope).

Array operations

The following definition has been executed in this page (access the console to manipulate it): 

let aList = [1, 4, 9, 16, 25]

Membership in arrays and strings

The method includes() checks if an element is in an array or if a string is in another string.

Mnemonic: The word in is in includes. 

arr.includes(3)
s.includes("and")
