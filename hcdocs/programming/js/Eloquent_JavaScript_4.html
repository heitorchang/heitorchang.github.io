1. Values, Types, and Operators

Strings

Strings that use single or double quotes behave very much the same.

Backtick-quoted strings are called template literals. They can be multi-line and can embed values inside ${}. 

`half of 100 is ${100 / 2}`

Unary Operators

typeof object_or_literal returns a string value naming the type of its argument.

Automatic Type Conversion

When null or undefined occurs on either side of ==, it produces true only if the other side is also one of null or undefined.

This behavior is useful to test whether a value has a real value instead of null or undefined, using either == or !=.

Short-circuiting of Logical Operators

Only the values 0, NaN, "" (empty string) are considered as false. [] is considered true. (Try [] || "abc" or if ([]) { ... }.

The ?? operator checks if the left-hand side value is null or undefined. If so, it returns the value on the right.

2. Program Structure

Bindings

The keyword let defines a binding. A binding grabs hold of a value. The = operator makes the name of the binding point to the value on the right-hand side.

Control Flow

Number(), String(), Boolean() convert their argument to those types.
