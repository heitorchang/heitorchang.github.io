Basic Svelte

Reactivity

State

To make a declaration reactive, wrap the value with $state(...). This is called a rune. It tells Svelte that count is not an ordinary variable. It's part of the Svelte language. 

let count = $state(0);

Deep state

State reacts to reassignments and mutations (this is deep reactivity). Deep reactivity is implemented with proxies.

The example is mutating an array: 

let numbers = $state([1, 2, 3]);

numbers[numbers.length] = 10;
numbers.push(20);

Derived state

Derived state is declared with the $derived rune. Unlike normal state, derived state is read-only. 

let numbers = $state([1, 2, 3]);
let total = $derived(numbers.reduce((t, n) => t + n, 0));

Inspecting state

$state.snapshot(...) creates a non-reactive snapshot of the state.

Also, the $inspect rune automatically logs a snapshot of the state whenever it changes. 

$inspect(numbers);
$inspect(numbers).with(console.trace);

Note: $inspect can only be used inside an effect (e.g. during component initialization).
