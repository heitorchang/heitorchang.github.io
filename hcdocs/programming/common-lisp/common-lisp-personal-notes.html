Emacs SLIME usage

Place cursor anywhere over a function definition or immediately after and type C-c C-c to compile it. It will be added to a kind of registry where M-. can be used to jump to a function's definition.

Loading a file also enables "jumping to the definition".

M-. jumps to definition M-, jumps back to where you were

To find usages (where a function is called)

C-c C-w C-c (slime-who-calls)

Debugging

When an error occurs, typing v in a debugger frame jumps to the source code location. The function must have been compiled first, though with C-c C-c

DEFVAR vs. DEFPARAMETER

(source: Practical Common Lisp)

Both DEFVAR and DEFPARAMETER create global variables. DEFPARAMETER always re-assigns the variable's value, but DEFVAR only does so for an undefined variable.

For example, *count* should be defined with DEFVAR because if the code is changed, you wouldn't want the count to be reset.

PLISTs (property lists)

(source: Practical Common Lisp)

(list :a 1 :b 2 :c 3) uses keyword symbols as property names.

To access a value:

(getf (list :a 1 :b 2 :c 3) :a)

To change a value:

(setf (getf plist :a) 100)

Objects made with QUOTE should not be modified

(source: Common Lisp HyperSpec: QUOTE)

The consequences are undefined if literal objects (including quoted objects) are destructively modified.

ASDF

The manual can be found at https://asdf.common-lisp.dev/asdf.html

To create a foo system:

File location

Create the ~/common-lisp/foo/foo.asd file:

(defpackage :com.heitorchang.foo-system (:use :asdf :cl))
(in-package :com.heitorchang.foo-system)

(defsystem "foo"
  :description "foo: a tutorial system"
  :version "0.0.1"
  :author "Heitor Chang"
  :license "MIT"
  :components ((:file "hello")))

and a ~/common-lisp/foo/hello.lisp file:

(in-package :com.heitorchang.foo-system)

(defun hello-foo () (format t "Hello FOO"))

To load:

(asdf:load-system "foo")

(in-package :com.heitorchang.foo-system)

CL-WHO

The project cl-heiapps includes a few common scenarios:

(defun html-header (title)
  (cl-who:with-html-output-to-string (dummy)
    (:div
     (:p (cl-who:str (format nil "Header: ~A" title))))))

(defun html-card (content)
  (cl-who:with-html-output-to-string (dummy)
    (:div :style "display: inline-block; border: 1px solid #ccc; padding: 1rem;"
          (:span (cl-who:str content)))))

(defmacro html-page (title &rest body)
  `(cl-who:with-html-output-to-string (dummy)
     (cl-who:str cl-who:*prologue*)
     (:html
      :lang "en"
      (:head
       (:meta :charset "UTF-8")
       (:meta :name "viewport" :content "width=device-width, initial-scale=1")
       (:title (cl-who:str ,title)))
      (:body
       ,@body))))

(easy-routes:defroute home "/" ()
  (let ((username (get-username)))
    (cl-who:with-html-output-to-string (dummy)
      (:div "Welcome to CL-Heiapps!"
            (if (string= username "")
                (cl-who:htm (:p
                             (:a :href "/login" "Login")))
                (cl-who:htm (:p
                             (:a :href "/user-home" "User home"))))))))

(easy-routes:defroute template "/template" ()
  (html-page
   "Title: Template"
   (cl-who:str (html-header "Template"))
   (:div "Body Content")))

(easy-routes:defroute reciprocal "/reciprocal/:n" ()
  (let ((value (parse-integer n)))
    (cl-who:with-html-output-to-string (dummy)
      (cl-who:str (html-header "Reciprocal"))
      (:div
       (:p (cl-who:str (html-card "Card: Reciprocal")))
       (:p (cl-who:str (format nil "Inverse of ~A is ~A" value (/ 1 (* 1.0 value)))))))))
