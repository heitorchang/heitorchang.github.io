Emacs SLIME usage

Place cursor anywhere over a function definition or immediately after and type C-c C-c to compile it. It will be added to a kind of registry where M-. can be used to jump to a function's definition.

C-c C-k will compile the entire file.

Loading a file also enables "jumping to the definition".

M-. jumps to definition M-, jumps back to where you were

To find usages (where a function is called)

C-c C-w C-c (slime-who-calls)

Set M-? to slime-who-calls in .emacs

Debugging

When an error occurs, typing v in a debugger frame jumps to the source code location. The function must have been compiled first, though with C-c C-c

DEFVAR vs. DEFPARAMETER

(source: Practical Common Lisp)

Both DEFVAR and DEFPARAMETER create global variables. DEFPARAMETER always re-assigns the variable's value, but DEFVAR only does so for an undefined variable.

For example, *count* should be defined with DEFVAR because if the code is changed, you wouldn't want the count to be reset.

PLISTs (property lists)

(source: Practical Common Lisp)

(list :a 1 :b 2 :c 3) uses keyword symbols as property names.

To access a value:

(getf (list :a 1 :b 2 :c 3) :a)

To change a value:

(setf (getf plist :a) 100)

Objects made with QUOTE should not be modified

(source: Common Lisp HyperSpec: QUOTE)

The consequences are undefined if literal objects (including quoted objects) are destructively modified.

hcutils

;;;; hcutils.asd
(defpackage :hcutils
  (:use :asdf :cl)
  (:export
   :join
   :with-stream-to-write
   :backup-dir))

(in-package :hcutils)

(defsystem "hcutils"
  :description "hcutils: shortcuts and utilities"
  :version "0.0.1"
  :author "Heitor Chang"
  :license "MIT"
  :depends-on (:str)
  :components ((:file "hcutils")))

;;;; hcutils.lisp
(in-package :hcutils)

(defun join (&rest names)
  "Concatenate names."
  (apply #'concatenate 'string names))

(defun md5sum (filename)
  "Compute the checksum of the file."
  (car (uiop:split-string (uiop:run-program (list "md5sum" (namestring (truename filename))) :output :string))))

(defmacro with-stream-to-write ((stream filename) &body body)
  "Pass a variable name to serve as the output stream (for example, out) and a filename.
The body may include calls such as (format out a-string)"
  (let ((fname (gensym "FILENAME"))
        (backup-fname (gensym "BACKUP-FILENAME")))
    `(let* ((,fname ,filename)
            (,backup-fname (format nil "~A.backup.~A" ,fname (get-universal-time))))
       (when (probe-file ,fname)
         (uiop:copy-file ,fname ,backup-fname))
       (ensure-directories-exist ,fname)
       (with-open-file (,stream ,fname
                                :direction :output
                                :if-exists :supersede)
         ,@body)
       ;; check if backup and newly created files are the same
       (when (and (probe-file ,backup-fname)
                  (equal (md5sum ,fname) (md5sum ,backup-fname)))
         (uiop:delete-file-if-exists ,backup-fname))
       t)))

Full example: basic-web-page

;;;; basic-web-page.asd
(defpackage :basic-web-page
  (:use :asdf :cl :hcutils)
  (:export :build-web-page))

(in-package :basic-web-page)

(defsystem "basic-web-page"
  :description "basic web page: HTML, CSS, JS"
  :version "0.0.1"
  :author "Heitor Chang"
  :license "MIT"
  :depends-on (:hcutils)
  :components ((:file "basic-web-page")))

;;;; basic-web-page.lisp
(in-package :basic-web-page)

(defparameter *web-root* "~/tmp_hcbel/basic-web-root/") ; add a trailing slash

(defun write-css ()
  (with-stream-to-write out (join *web-root* "style.css")
    (format out "
body {
  font-family: sans-serif;
  background-color: #cfd;
  color: #228;
}")))

(defun write-script ()
  (with-stream-to-write out (join *web-root* "script.js")
    (format out "
console.log('Written by basic-web-page.lisp');
")))

(defun write-home-page ()
  (with-stream-to-write out (join *web-root* "index.html")
    (format out "<!DOCTYPE html>
<html>
  <head>
    <link rel='stylesheet' href='style.css'>
  </head>
  <body>
    <h1>Welcome to my site!</h1>
    <script src='script.js'></script>
  </body>
</html>")))

(defun build-web-page ()
  (write-css)
  (write-script)
  (write-home-page))

To run:

(asdf:load-system "basic-web-page")
(basic-web-page:build-web-page)

CL-WHO

The project cl-heiapps includes a few common scenarios:

(defun html-header (title)
  (cl-who:with-html-output-to-string (dummy)
    (:div
     (:p (cl-who:str (format nil "Header: ~A" title))))))

(defun html-card (content)
  (cl-who:with-html-output-to-string (dummy)
    (:div :style "display: inline-block; border: 1px solid #ccc; padding: 1rem;"
          (:span (cl-who:str content)))))

(defmacro html-page (title &rest body)
  `(cl-who:with-html-output-to-string (dummy)
     (cl-who:str cl-who:*prologue*)
     (:html
      :lang "en"
      (:head
       (:meta :charset "UTF-8")
       (:meta :name "viewport" :content "width=device-width, initial-scale=1")
       (:title (cl-who:str ,title)))
      (:body
       ,@body))))

(easy-routes:defroute home "/" ()
  (let ((username (get-username)))
    (cl-who:with-html-output-to-string (dummy)
      (:div "Welcome to CL-Heiapps!"
            (if (string= username "")
                (cl-who:htm (:p
                             (:a :href "/login" "Login")))
                (cl-who:htm (:p
                             (:a :href "/user-home" "User home"))))))))

(easy-routes:defroute template "/template" ()
  (html-page
   "Title: Template"
   (cl-who:str (html-header "Template"))
   (:div "Body Content")))

(easy-routes:defroute reciprocal "/reciprocal/:n" ()
  (let ((value (parse-integer n)))
    (cl-who:with-html-output-to-string (dummy)
      (cl-who:str (html-header "Reciprocal"))
      (:div
       (:p (cl-who:str (html-card "Card: Reciprocal")))
       (:p (cl-who:str (format nil "Inverse of ~A is ~A" value (/ 1 (* 1.0 value)))))))))

UIOP shell commands

Interactive directory commands have the same name as Python's os. uiop:getcwd and uiop:chdir.

To run a command and view output: (uiop:run-program '("ls" "-l") :output :string).

Ignore errors (such as ls not finding a file)

(uiop:run-program '("ls" "-l" "abc.txt") :output :string :ignore-error-status t)
