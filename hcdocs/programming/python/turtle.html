Python Turtle

Tested in Windows 11

Initializing

In a REPL, type: 

import turtle
turtle.getscreen()
t = turtle.Turtle()

Resetting 

t.reset()

Moving 

t.up() # raise pen (nothing gets drawn while moving)
t.down() # lower pen (turtle movement draws a line)

t.home() # return to (0, 0)
t.goto(x, y)

Drawing 

t.color(color_name)
t.color(r, g, b)
t.color(pen_color, fill_color)
t.color((r1, g1, b1), (r2, g2, b2))

t.dot(n)
t.circle(r)

t.forward(n)
t.fd(n)

t.backward(n)
t.back(n)
t.bk(n)

t.left(deg)
t.lt(deg)

t.right(deg)
t.rt(deg)

Writing strings 

t.write(s, move=False, align='left, font=('Arial', size, 'normal'))
