Django Minimal App

Source: Django tutorial (v5.0) and William Vincent's Django for Professionals (v4.0)

Creating a new project

cd ~/code/venvs/

python3 -m venv venvdj50
source ~/code/venvs/venvdj50/bin/activate

pip install --upgrade pip
pip install django~=5.0

cd ~/code/django_projects/

# it's best to leave out special characters in the app name
# and they cannot be the same as built-in names
django-admin startproject minimalapp

cd minimalapp/

Create a .gitignore file

__pycache__/
mysecret*
db.sqlite3

Remove the SECRET_KEY from the settings file

Create a file /minimalapp/mysecrets.py (in the same directory as settings.py). 

SECRET_KEY = "...some random string..."

In settings.py, remove SECRET_KEY and add: 

from .mysecrets import SECRET_KEY

Create a custom user model

Run: 

python manage.py startapp accounts

Add to accounts/models.py (keep the existing code) 

from django.contrib.auth.models import AbstractUser


class CustomUser(AbstractUser):
    pass

Add to settings.py: 

INSTALLED_APPS = [
    ...
    'accounts',
]

...

AUTH_USER_MODEL = 'accounts.CustomUser'

Run: 

python manage.py makemigrations accounts
python manage.py migrate

Add the custom user to the Admin site

Add to accounts/admin.py (keep the existing code) 

from django.contrib.auth import get_user_model

CustomUser = get_user_model()
admin.site.register(CustomUser)

Create a super user

Run: 

python manage.py createsuperuser

The email can be anything, like a@a.com. A possible username is admin.

Run the development server

Run: 

python manage.py runserver

To use another port number: 

python manage.py runserver 8080
