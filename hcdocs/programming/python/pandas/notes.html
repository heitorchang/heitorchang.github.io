Basic data structures

  • Series: a one-dimensional labeled array
  • DataFrame: a two-dimensional data structure (2-D array or table with rows and columns). Each column may have a different dtype.

s = pd.Series([1, 3, 5, np.nan, 6, 8])

Converting to the NumPy representation of the data

DataFrame.to_numpy() will convert the underlying data to an array of arrays with a single dtype that will hold all the dtypes of the Pandas DataFrame.

Although Pandas DataFrames can have different dtypes for each column, the NumPy array will have only one dtype.

Selection

For production code, use the following optimized methods:

  • DataFrame.at(): used if you only need to get or set a single value (by label).
  • DataFrame.iat(): access a single value by integer position
  • DataFrame.loc(): access a group of rows and columns by label(s). Unlike Python slices, both endpoints are included.
  • DataFrame.iloc(): access a group of rows and columns by integer position(s). In this case, the right endpoint is not included. For example, df.iloc[:, 0:1] returns the first column only.

iloc

df.iloc[:, [1]] # selects the second column
