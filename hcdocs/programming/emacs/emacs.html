GNU Emacs

Verifying source code file signature

Download https://ftp.gnu.org/gnu/gnu-keyring.gpg and run: 

gpg --import gnu-keyring.gpg

Use the signature from https://ftp.gnu.org/gnu/ (in theory, a mirror could put whatever signature file it wishes).

Then run gpg --verify emacs-xx.x.tar.xz.sig

Switch to JetBrains Mono font

Swap the font line in .emacs and my-tango-theme.el with: 

'(default ((t (:inherit nil :stipple nil :background "PapayaWhip" :foreground "black" :inverse-video nil :box nil :strike-through nil :overline nil :underline nil :slant normal :weight light :height 110 :width normal :foundry "outline" :family "JetBrains Mono Light"))))

Hi-lock Mode

Allows highlighting occurrences of words (symbols) 

M-x hi-lock-mode
C-x w     .  ;; highlights symbol at point
C-u C-x w r  ;; unhighlights everything

Packages

Run:

  1. M-x package-refresh-contents
  2. M-x package-install

Enable debug messages

When an error occurs, it may be helpful to get more information. Run M-x toggle-debug-on-error

Lisp mode

In the REPL, type C-j to create a newline.

Japanese ん

Type n' to write ん.

Type z- to write 〜.

In the japanese input method, type K to transform the word into katakana.

Emacs built-in current time function

(setenv "TZ" "UTC0")
(format-time-string "%Y-%m-%d %H:%M" (current-time))

Slowness in shutting down due to auto-complete

A Stack Overflow search revealed that after a long time of usage, ac-comphist.dat becomes bloated and slows down shutdown. Simply move it to a .bak file or remove it.
