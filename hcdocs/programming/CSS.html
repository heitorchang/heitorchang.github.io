CSS Notes

A problem I was having while designing this site (hcdocs) was the border of pre blocks. The simple way creates a border that moves with the viewport size.

Two solutions were found with the use of ChatGPT:

First, setting display: inline-block; makes the border follow the length of the longest line inside the pre. 

pre {
  display: inline-block;
  box-sizing: border-box;
  margin: 1rem;
  padding: 1rem;
  border: 1px solid #ccc;
  background-color: #eee;
  border-radius: 0.8rem;
  overflow: visible;
  min-width: calc(100% - 2rem);
}

A cleaner solution is to use a horizontal scrollbar with overflow: auto;. (The surrounding text won't scroll when you want to read the block's content). 

pre {
  display: block;
  box-sizing: border-box;
  margin: 1rem;
  padding: 1rem;
  border: 1px solid #ccc;
  background-color: #eee;
  border-radius: 0.8rem;
  overflow: auto;
  min-width: calc(100% - 2rem);
}
