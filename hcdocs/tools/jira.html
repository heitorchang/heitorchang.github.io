Jira

Sprint board

For a ticket to appear on the sprint board:

  • Click on "Backlog" on the left sidebar
  • Make sure the issue to send to Sprint has a component and "Fix version" equal to "Sprint N" (N is the sprint number)
  • Right-click on an issue in the bottom half (Backlog), to send to Sprint N

Search with a Query

Click on "Filters" on the top toolbar, then "Advanced issue search"

Write a query such as project = "MY_PROJECT" and fixVersion = "Sprint 3"

Use "View all filters" to see examples of other queries.

There is "ORDER BY property ASC/DESC"
