Ubuntu notes

Note: Currently using 23.04

Creating a .desktop file

[Desktop Entry]
Name=Emacs29
Exec=/home/heitor/bin/emacs-29.1
StartupNotify=false
Terminal=false
Type=Application
Icon=emacs

Adding a custom shortcut to the dock

  1. Create a .desktop file (see above for Emacs 29) in ~/.local/share/applications/
  2. Click on Activities on the top bar. Search for the name given in the .desktop file.
  3. Right-click and choose "Add to Favorites"
