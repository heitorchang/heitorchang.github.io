Ubuntu notes

Note: Currently using 23.04

Creating a emacs.desktop file

[Desktop Entry]
Name=Emacs29
Exec=/home/heitor/bin/emacs-29.1
StartupNotify=false
Terminal=false
Type=Application
Icon=/path/to/imagefile.png_or_jpg

Adding a custom shortcut to the dock

  1. Create a emacs.desktop file (see above for Emacs 29, replace "emacs" with the needed program's name) in ~/.local/share/applications/
  2. Click on Activities on the top bar. Search for the "Name" given in the emacs.desktop file.
  3. Right-click and choose "Add to Favorites"

Disable Ctrl + Alt + Left/Right

These keyboard shortcuts are used in Emacs Paredit, so to disable them, go to Settings > Keyboard > Keyboard Shortcuts. Search for "workspace" and disable all of them. Although Ctrl + Alt + Left/Right does not appear, disabling them solves the issue.

Disable Ctrl + Alt + D

Search "Hide all normal windows" in Keyboard shortcuts and disable it.

International keyboard for Portuguese

In Ubuntu, choose "Intl. with dead keys". In Chrome OS, choose International PC Keyboard.

Type Right Alt + comma for a cedilla (รง).
