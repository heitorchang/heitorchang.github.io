PyCharm Community

Downloaded Version 2023.2.1.

Shortcuts

  • Ctrl + Alt + s: settings
  • Shift - Shift: search
  • Ctrl + Shift + i: go to definition
  • Alt + Shift + 7: find usages
  • Alt + Shift + left/right: navigate to previous/next location
  • Ctrl + Alt + i: indent
  • Ctrl + Shift + f: find all

Customization

  • Disable "Choose Lookup Item Replace" in keymap, and set "Tab" to "Choose Lookup Item". This change prevents the tab autocomplete from deleting the text ahead of it.
  • Disable autosave (System settings)
  • Remove inlay hints
  • (Keymap) set F10 to "Save all (Ctrl + s)
  • (Keymap) set Alt + o for "Goto next splitter".
  • (Keymap) set F12 to "Split right".
  • (Keymap) set F9 to "Run File in Python Console".
  • (Keymap) set F8 to "Unsplit all".
  • (Keymap) set Ctrl + Right Click to "Open in opposite group".
  • (Keymap) set Alt + C to "Tool Windows > Python Console".
  • Enable soft-wrap for *.py and uncheck "Only show indicators for the current line".

Change typing mismatches from warning to error:

Search for "incorrect type" in Editor > Inspections, change severity to Error.

Run script in console (keep interactive prompt after running script)

  1. Next to the "Run" icon on the top toolbar, click on "Edit Configurations".
  2. Under "Run script", set the script name, and set the working directory to a valid location.
  3. Click on Modify options and add "Run with Python Console"
  4. You may need to click on the Python logo on the bottom left to show the Python Console.
  5. Re-run after making changes.

Python or Debug console

The simplest way to interactively test a file's code is to right click and choose "Run File in Python Console". It does not have a keyboard shortcut assigned to it, so I picked F9.

Highlight code and type Alt + Shift + E to send to console.

If nothing is highlighted, the line under the cursor is sent.

Type Esc to return to the editor.

There doesn't seem to be a way to change the behavior of the console to go back in its command history other than pressing Up (which will scroll through a multi-line command). This is not very user-friendly.

Miscellaneous

Check line separators under "Editor > Code Style" then on the right panel, General.

Disable hiding imports under "Code Folding" > General

Comparing files (diff)

Click and select two files on the main menu (file browser) and type Ctrl + d
