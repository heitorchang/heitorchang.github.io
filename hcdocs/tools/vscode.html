Visual Studio Code

Interactive Python development

Requires installing ipykernel.

In an open Python file, click on a triangular "Play" button to the right of the file's tab (hovering over it says "Run current file in interactive window")

After changing the Python source, clicking on this button again will re-evaluate the code.

Keybindings

Click here to view the latest keybindings.

The file's location in Windows is c:/Users/USERNAME/AppData/Roaming/Code/User/keybindings.json

HTML / JavaScript Live Preview

There are many "Live server" extensions but I prefer the (official) Microsoft's "Live Preview" (it has the Microsoft badge under the extension name) despite it still being Experimental.

In the Extensions sidebar, click on the gear icon to access its Settings. Then, change "Auto Refresh Preview" to "On Changes to Saved Files."
