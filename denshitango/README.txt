denshitango

A webpage that displays flash cards for Japanese words.

Data structures

The word database is a list of lists. Each element has this structure:

["category", "kana pronunciation", "kanji", "english suggestion", book_id]

The book_id corresponds to an audio filename.

Content

The full set of words and audio includes copyrighted material and will not be distributed here.

The file words.js contains only 6 of these words.
