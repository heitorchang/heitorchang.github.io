Code editor

Tutorial

This website contains a bare-bones Python interpreter in the bottom panel that processes textual input just like a standard Python installation.

Due to the limitations of being inside a single page of a web browser, it cannot handle graphics, network connections, operating system functions, or similar actions like a full Python installation in your computer.

Temporary code may be typed on the left panel.

Keyboard shortcuts

Ctrl + Enter Evaluate the code block, without clearing the inputted code
Alt + Enter Evaluate the code block and clear the inputted code
Ctrl + Up/Down Retrieve previously evaluated code
Esc Clear the code editor's contents
Alt + J Focus on the code editor
Alt + K Focus on the Python interpreter

Throughout the book, there will be several code blocks. Clicking on one of them will transfer the code to the code editor. This action will replace whatever was there, so be careful! 

def square(x):
    return x * x

After defining the square function, it may be called to compute the result. Clicking below will print the result in the interpreter panel. 

print(square(99))
